To customize your favicon (browser tab icon):

Option 1 - Replace the SVG (recommended):
1. Edit static/favicon.svg
2. Change the "N" to your initial or create a custom design
3. Adjust colors if desired (current: #15284b background, #48a9a6 letter)

Option 2 - Use a PNG/ICO file:
1. Create or obtain a square image (at least 32x32 pixels, preferably 512x512)
2. Convert it to .ico or .png format
3. Name it "favicon.ico" or "favicon.png"
4. Place it in the /static/ directory

The SVG favicon will automatically appear in browser tabs once deployed.
Current placeholder shows "N" for Nicolau.

You can use online tools like:
- https://favicon.io/ (generate from text, image, or emoji)
- https://realfavicongenerator.net/ (comprehensive favicon generator)
